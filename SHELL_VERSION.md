# agile-board-cli

## Install
download the codebase and get the sh directory path


## How to use
### config account info
```
 sh init-account.sh
```

### Move card 
#### move to inDev 
```
sh auto-move-to-indev.sh
```
*recommend add this to commit hook for git
```
sh add-commit-msg-hook.sh
```

#### move to devDone
```bash
sh dev-done.sh
```
